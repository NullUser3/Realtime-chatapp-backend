# Realtime Chat App - Backend

This is the backend API for the Realtime Chat App.  
It handles authentication, database operations, and realtime messaging with WebSockets.

## Tech Stack

- Node.js
- Express
- MongoDB
- Socket.io
- JWT Authentication
- Rate Limiting & Security Middleware

## Features

- User authentication & authorization
- Realtime messaging
- Typing indicators
- Online/offline status
- Search users
- Rate limiting for security
